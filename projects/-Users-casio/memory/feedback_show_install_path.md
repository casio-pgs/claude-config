何かをインストールまたはダウンロード・配置したとき、インストール先のフルパスを必ず表示する。

**Why:** どこに入ったか把握しておきたいため。

**How to apply:** curl/brew/pipなどでファイルを置いたり、インストールしたりした直後に「インストール先: /path/to/...」の形で明示する。

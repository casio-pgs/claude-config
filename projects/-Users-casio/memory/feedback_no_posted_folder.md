Obsidianにファイルを保存する際、`_posted` フォルダを作成しない。指定されたフォルダに直接保存する。

**Why:** ユーザーが意図しない `_posted` フォルダが作られて混乱した。

**How to apply:** Obsidianへの保存時は常に指定フォルダ直下に直接ファイルを作成する。
